☁️ Google Drive Sync Setup
⚠️ One-time 5-minute setup. Your tracker is hosted at https://sperezjr-byte.github.io/tracker — no Terminal command needed. Once connected, data auto-saves to Google Drive every time you make a change, from any device.
1
Go to Google Cloud Console
Open console.cloud.google.com and sign in with the Google account you want to save data to.
2
Create a New Project
Click the project dropdown at the top → New Project → name it Top Tier Tracker → click Create.
3
Enable Google Drive API
Left menu → APIs & Services → Library → search Google Drive API → click it → click Enable.
4
Configure OAuth Consent Screen
Left menu → OAuth consent screen → choose External → click Create.

• App name: Top Tier Tracker
• User support email & Developer contact: your Gmail

Click Save and Continue through all steps. Under Test users, add your own Gmail address. Click Save.
5
Create OAuth Credentials
Credentials → Create Credentials → OAuth client ID

• Application type: Web application
• Name: Top Tier Tracker
• Authorized JavaScript origins → Add URI: https://sperezjr-byte.github.io

Click Create. Copy the Client ID from the popup — it looks like:
123456789-abc...apps.googleusercontent.com
6
Paste Your Client ID Below
Paste the Client ID from step 5 here, then click Connect.
